# Four Rules I Use When Explaining Oncology

Explaining oncology to different audiences is not about simplifying until meaning disappears. It is about preserving accuracy while managing complexity.

These are the four rules I keep in every talk.

## Rule 1 — Start with context, not terminology

People process relevance before definitions.

Instead of opening with molecular details, I start with:

"What problem are we trying to solve for patients and teams?"

## Rule 2 — Define one layer at a time

I avoid stacked definitions. First concept, then mechanism, then implication.

When all layers arrive at once, audiences remember none.

## Rule 3 — Name uncertainty explicitly

Trust increases when uncertainty is visible.

I use phrases like:

- "current evidence suggests"
- "this signal is promising but not definitive"
- "these findings need larger validation"

## Rule 4 — End with action

Every explanation should finish with a practical next step:

- what to monitor,
- what to discuss,
- what to test next.

Without action, good explanations become passive information.

---

Clarity in oncology is not a soft skill. It directly affects decisions.
